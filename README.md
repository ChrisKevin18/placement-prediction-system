# Placement Prediction System

## Project Overview
This project predicts whether a student will be placed based on academic and skill-related features using Machine Learning.

The system performs:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training using Random Forest
- Model Evaluation
